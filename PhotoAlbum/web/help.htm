Help

This is a simple-to-use photo album application written in JavaScript and JSP.

  • Navigate through the pages with the and buttons.
  • Use to go to the first page and to go to the last page.
  • Double click an image for a view of the comments.
  • You may search the images using the box on the top of every page. The search matches photo's titles and descriptions.
  • Click 'Login' on any page and enter your credentials. You will be able to comment on photos to which you have been granted access.
